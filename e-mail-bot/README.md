## API для генерации коммерческих предложений
Доступен новый эндпоинт для генерации коммерческих предложений: {{API_URL}}/gpt_templater_v2

Эндпоинт принимает POST запросы с данными о товаре/услуге и генерирует профессиональное коммерческое предложение.

    "type1": "Говядина",              # тип продукции 1
    "type2": "вырезка",               # тип продукции 2

    "unit": "кг",                     # единица измерения
    "user_company_name": "Компания",  # название компании

## Важная информация
При отправке запроса к API необходимо устанавливать время на ответ (timeout) **не менее 60 секунд**, т.к. стандартных 30 секунд может быть не достаточно.

В случае, если пользователь, отреагировавший на рассылку, не зарегистрирован на meatinfo, в теле запроса необходимо передавать `"user_id": 0`.

## Принцип работы
1. Для полученных из json id пользователей, выполняется запрос к сервису user_card, для получения дополнительной информации о пользователях.
1. Из полученных данных и массива сообщений из json генерируется текст быстрого сообщения с помощью Chat GPT.
