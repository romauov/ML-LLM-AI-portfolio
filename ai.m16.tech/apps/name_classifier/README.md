# История улучшения модели и обнаруженные ошибки

Пример обращения к модели классификатора имени в виде api:
(https://ai.m16.tech/api/name_classifier?name=регина)
результат: 
{"name": "Регина", "date": "2023-03-09, 11:15:55", "label": 1}

Пример обращения к модели классификатора фамилии в виде api:
(https://ai.m16.tech/api/surname_classifier?name=дубина)
результат: 
{"surname": "дубина", "date": "2023-03-09, 11:32:31", "label": 1}


Модель 09-01:  
- Результат оценки 98.5%, допускает ошибки в именах Сергей, Надежда, Любовь, Роман 

Модель 19-01: 
- Результат оценки 99.3%, модель может пропускать опечатки, не допускает пробелов в имени, фильтрует мат. Уверенно работает с русским языком, может невпопад предсказать имена на латинице  

Модели 09-03:
- Построена на основе рекуррентных и сверточных сетей
- Модель name классифицирует имена, модель surname - фамилии
