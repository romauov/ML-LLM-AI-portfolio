## Требования к данным
Данные должны включать в себя:
1. столбец даты: день, месяц, год
2. столбец, который нужно спрогнозировать
3. Колонки должны быть в первой строке файла


## Описание работы Pipeline

Pipeline (pipeline.py) включает в себя:

1. Чтение данных
2. Поиск гиперпараметров (tuner.py)
3. Валидация результатов модели (forecasting_visualization.py)
4. Обучение ансамбля (ensemble.py)
5. Визуализация прогноза

При добавлении нового блока в pipeline:
1. Добавить блок в pipeline (pipeline.py)
2. Добавить соответствующий вывод в gradio.py
