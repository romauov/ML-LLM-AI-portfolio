# Создание цифровых карточек пользователя
### API находится по адресу: `https://api.ai.m16.tech/api/user_card`

## Данные на вход
Данный API принимает `json`.
```json
{
    "user_id": 52768
}
```
## Структура ответа
```json
{
    "card": "Тут будет карточка пользователя"
}
```

Карточка в формате `str`, который можно сохранить в файл `txt` при желании.

## Возможные ошибки

```json
{"Ошибка": "Такого пользователя нет в базе"}
```

```json
{"Ошибка": "Ключ user_id в полученных данных не найден"}
```
