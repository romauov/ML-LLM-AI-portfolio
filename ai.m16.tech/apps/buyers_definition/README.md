## Модель покупатель-продавец на основе случайного леса

На вход конвейеру подается информация о продукте, времени с последнего действия, количество пользователей, которое нужно для рассылки 

Пример запрос: {"products": "Говядина", "datePeriod": 90, "numberOfUsers": 1}

Результат: [{"email": "ataljants@mail.ru"}]

Во время подготовки результата отбрасываются пользователи, которые не читают рассылки (открыли меньше 10% полученных предложений)

Среднее время возврата результат: 40 секунд

Для обращения к модели: /api/buyers-users
