# Способ взаимодействия
## API

Api доступно ссылке: https://ai.m16.tech/api/card_recommendations

Авторизация в ai.m16.tech:

Логин: user

Пароль: {{API_PASSWORD}}

Принимает: 
```json
{  
    "tsop_id": "51",
    "number_of_users": 100,
    "site":"meatinfo",
    "user_portrait":"{'prods': 'Форель охлажденная', 
                      'info': ' Компания занимается переработкой рыбы, переработчик.,
                      'position': 'менеджер по закупкам, руководитель отдела закупок.'}"}'
```
Возвращает: 
```json
{
    "Emails": ["123@mail.ru"]
}
```

# Описание работы Pipeline

Принцип работы построен на поиске похожих карточек, векторы получены при помощи LLM.

Pipeline состоит из нескольких основных компонентов:
1. Векторизация суммаризированной карточки
2. Получение пользователей и эмбеддингов полученных из их карточек
3. Обучение faiss-индекса считанными эмбеддингами
4. Агрегация данных


Используемые данные:
1. Резюмированные карточки клиентов с 2017 года
2. tradeboard
