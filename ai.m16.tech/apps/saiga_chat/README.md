# Описание
Сервис-чат с ассистентом Сайгой(Saiga-7b-lora). Модель была дообучена на собранном датасете с коммерческими предложениями.

Код лежит на вируталке 3090, там же развернут сервер для обращения к сайге. 

# Api
Api доступно ссылке: (https://ai.m16.tech/api/saiga_chat)

```json
Принимает: 
{
   "text": "Привет. Как дела?",
   "temperature": 0.6
}
```

Возвращает:

```json
Принимает: 
{
   "answer": "Здравствуйте! я пишу коды для интеллектуальных ассистентов.",
}
```
## Сurl запрос

```sh
curl -u "{{API_USER}}:{{API_PASSWORD}}" -d '{"text": "Привет. Как дела?", "temperature": 0.6}' -H "Content-Type: application/json" -X POST https://ai.m16.tech/api/saiga_chat
```
