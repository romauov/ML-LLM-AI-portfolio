## Описание модели
Модель квалификатора лидов используется для классификации сообщений пользователю, обучена на 2 датасетах: ответы на рассылки и цепочки сообщений из чата. Для ускорения работы микросервиса количество выполняемых операций подготовки текста было снижено. Специально разработана по гранту ЦОП. Отсекает ненужные сообщения, позволяет автоматизировать отправку запрашиваемой покупателем информации, что экономит время менеджера на рутинные ответы.

## Описание модели

Модель квалификатора построена на основе сверточных слоев и использует инференс onnxruntime.

Способ подготовки текста:
1. Усечение последовательности до 150 токенов
2. Удаление всех тегов
3. Кодирование токенов по словарю

Модель обучена на 7 классах:
* встречное предложение (комплементарное)
* встречное предложение (некомплементарное)
* запрос конкретной продукции 
* запрос прайса
* отказ 
* спам/неинформативно
* уточнение информации (в этот класс добавлены запрос фото, запрос контакта)
