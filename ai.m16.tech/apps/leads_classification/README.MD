# Leads Classification

Проект `leads_classification` является системой на основе машинного обучения, которая классифицирует лиды по их содержанию на различные категории.

## Авторы

* Колташев Ярослав
* Авзалов Дмитрий

## Данные

Данные, на которых обучалась модель `spacy` это выгрузка из двух проектов Label studio. 

1. [Выгрузка из писем](http://labview-robot-cv.inline-ltd.ru/projects/11/data?tab=19)
2. [Выгрузка из диалогов ](http://labview-robot-cv.inline-ltd.ru/projects/13/data?tab=24)

Данные, на которых обучалась модель `bert` это выгрузка из [выгрузка из диалогов ](http://labview-robot-cv.inline-ltd.ru/projects/13/data?tab=24). 

## Обучение
Обучение проходило на платформе Kaggle. 

Ссылка на проект Spacy: [Проект textcat-spacy](https://www.kaggle.com/code/yarasslove/textcat-spacy)

Ссылка на проект Bert: [Проект bert](https://www.kaggle.com/code/dmitryavzalov/bert-meat-messages/notebook)

## Список категорий

Классы для модели Spacy

* спам
* запрос конкретной продукции
* встречное предложение (некомплиментарное)
* Не могу опеределить класс
* встречное предложение (комплиментарное)
* уточнение информации
* запрос прайса

Классы для модели Bert

* запрос конкретной продукции
* встречное предложение (некомплиментарное)
* встречное предложение (комплиментарное)
* уточнение информации
* запрос прайса

## Для разработчика

API проекта находится по адресу [https://ai.m16.tech/api/predict_leads](https://ai.m16.tech/api/predict_leads)

Демо проекта можно посмотреть [https://ai.m16.tech/api/leads_classification](https://ai.m16.tech/api/leads_classification)

API принимает json: `{'data': ['Продам гараж', 'Продам свинину 1кг 100 рублей'], 'model': 'spacy'}`

Параметр модель должен быть одним из следующих: `spacy`, `bert`.

Headers: 
* `Authorization: Basic dXNlcjo1S3lPUDdpUGtx`
* `Content-Type: application/json`
