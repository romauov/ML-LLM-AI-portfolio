## Описание:

Алгоритм для получения значений, характеризующих внимание человека. 
- Входные данные: ссылка на изображение
- Выходные данные: угол поворота головы по оси x, y, а также коэффициент открытия глаз

Изображение должно быть размещено в облачном хранилище Яндекс Диск. Доступ к изображению должен быть открыт.

В алгоритме используется модель для разметки лицевых точек из библиотеки MediaPipe. На основе полученных точек происходит вычисление углов поворота головы и коэффициента, характеризующего степень открытия глаз. 

Протестировать можно по следующей ссылке: https://ai.m16.tech/api/mediapipe_based_head_controller?image_path=https://disk.yandex.ru/i/gzOQr0EMeagXtw
где https://disk.yandex.ru/i/gzOQr0EMeagXtw - путь к изображению на яндекс диске.

Точность алгоритма сильно зависит от параметров камеры, с которой было сделано изображение. Для получения максимально точного результата следует производить калибровку камеры средствами библиотеки OpenCV, однако на данный момент функционал для калибровки камеры отсутствует.
