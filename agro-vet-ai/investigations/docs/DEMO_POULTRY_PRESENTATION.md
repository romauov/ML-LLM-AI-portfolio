## Сценарий 1: Респираторная проблема у бройлеров (10-15 мин)

### Описание кейса
Птицефабрика, бройлеры 14 дней, появились признаки респираторного заболевания: чихание, конъюнктивит, хрипы при дыхании. Падеж 2%/день. Вакцинация проводилась (инфекционный бронхит в суточном возрасте, спрей).

### Шаги демонстрации

**Шаг 1: Создание чата и начало расследования**

Открыть новый чат с агентом "VetRetro - Птицеводство"

**Первое сообщение:**
```
У нас проблема на птицефабрике. Бройлеры Кобб-500, возраст 14 дней, в птичнике №3.
Появились признаки респираторного заболевания: чихание, конъюнктивит, хрипы при дыхании.
Падеж около 30-40 голов в день из 3000 (примерно 1-1.5%/день).
Вакцинация проводилась: инфекционный бронхит в суточном возрасте (спрей).
```

**Ожидаемый результат:**
- Агент представится как специалист по птицеводству
- Задаст вопросы о микроклимате (аммиак, температура, вентиляция)
- Запросит информацию о других птичниках (системная vs локальная проблема)
- Проактивно начнет поиск в базе знаний по респираторным проблемам птицы

**Шаг 2: Предоставление данных о микроклимате**

**Второе сообщение:**
```
Микроклимат: температура 22°C (норма для возраста), влажность 65%.
При входе в птичник чувствуется запах аммиака - глаза немного щиплет.
Подстилка влажная в некоторых местах, особенно под поилками.
Вентиляция работает, но кажется недостаточной.
В других птичниках (№1 и №2) проблем нет, там птица младше (7 дней).
```

**Ожидаемый результат:**
- Агент идентифицирует высокий аммиак как предрасполагающий фактор
- Предложит провести вскрытие для дифференциальной диагностики
- Сформирует предварительные гипотезы (Mycoplasma + E.coli, IBV)
- Порекомендует срочные меры по улучшению вентиляции

**Шаг 3: Результаты вскрытия**

**Третье сообщение:**
```
Провели вскрытие 3 павших птиц.
Патология: воздухоносные мешки мутные, утолщенные стенки, небольшое количество казеозного экссудата.
Легкие частично уплотнены. Трахея с гиперемией, слизистый экссудат.
Синусы с небольшим отеком.
```

**Ожидаемый результат:**
- Агент распознает аэросаккулит (типично для Mycoplasma + E.coli)
- Порекомендует ПЦР на Mycoplasma gallisepticum и бакпосев
- Предложит антибиотикотерапию (тилозин, энрофлоксацин)
- Подчеркнет необходимость коррекции микроклимата ПАРАЛЛЕЛЬНО с лечением

**Финал:**
- Агент создаст структурированные файлы расследования
- Сформирует четкие рекомендации по лечению и профилактике
- Процитирует источники из базы знаний (Practical guide to broiler health management)

## Сценарий 2: Диарея и некротический энтерит у бройлеров (10-15 мин)

### Описание кейса
Бройлеры 21 день, внезапное повышение смертности (4%/день), темный дегтеобразный помет, снижение привесов.

### Шаги демонстрации

**Шаг 1: Описание проблемы**

**Первое сообщение:**
```
Птицефабрика, бройлеры 21 день.
Вчера резко вырос падеж - с обычных 10-15 голов/день до 120 голов/день (из 3000, это 4%!).
Помет у многих птиц темный, почти черный, дегтеобразный.
Птица угнетенная, перо взъерошено. Привесы упали.
Корм сменили 5 дней назад - перешли с стартера на гровер.
```

**Ожидаемый результат:**
- Агент сразу заподозрит некротический энтерит (Clostridium perfringens)
- Запросит информацию о кокцидиостатиках в корме
- Попросит провести срочное вскрытие
- Проактивно начнет поиск в базе знаний по некротическому энтериту

**Шаг 2: Дополнительные данные**

**Второе сообщение:**
```
Кокцидиостатиков в корме НЕТ - решили обойтись без них в этом туре.
Провели вскрытие: тонкий кишечник растянут газами, стенка дряблая.
На слизистой видны некротические бляшки/пленки серо-желтого цвета.
Содержимое кишечника темное, с неприятным запахом.
```

**Ожидаемый результат:**
- Агент подтвердит диагноз "некротический энтерит"
- Объяснит связь: отсутствие кокцидиостатиков → кокцидиоз → повреждение слизистой → Clostridium perfringens
- Порекомендует немедленное лечение амоксициллином/ампициллином
- Предложит микроскопию помета на кокцидии
- Даст рекомендации по профилактике (вернуть кокцидиостатики)

**Финал:**
- Конкретные дозы антибиотиков (с учетом птицы!)
- Рекомендации по коррекции программы кормления
- Долгосрочная профилактика для следующих туров

## Ключевые моменты для презентации

### Что показать ветеринарам:

1. **Специализация агента:**
   - Агент использует птицеводческую терминологию (бройлеры, несушки, престартер/стартер/гровер)
   - Знает специфику птицеводства (аммиак критичен, кокцидиоз, аэросаккулит)
   - Ищет информацию в книге "Practical guide to broiler health management"

2. **Проактивность:**
   - Сам ищет информацию в базе знаний
   - Задает правильные вопросы о микроклимате, вакцинации, кормлении
   - Создает структурированные файлы расследования

3. **Системный подход:**
   - Анализ микроклимата → вскрытие → лабораторная диагностика → лечение + профилактика
   - Учитывает многофакторность проблем
   - Дает конкретные рекомендации с дозами

4. **Практическая ценность:**
   - Все рекомендации со ссылками на источники
   - Конкретные дозы препаратов
   - Чек-листы для полевых осмотров
   - ROI расчеты (если потребуется)

### Вопросы, которые могут задать ветеринары:

**Q: А можно добавить инструкции по другим проблемам птицы?**
A: Да! Система уже поддерживает добавление новых инструкций. После обратной связи от вас мы можем создать инструкции по:
- Кокцидиозу (детальный протокол)
- Колибактериозу
- Ньюкаслской болезни
- Инфекционному бронхиту
- Проблемам несушек (снижение яйценоскости, каннибализм)

**Q: Будет ли агент давать рекомендации по вакцинации?**
A: Да, агент учитывает вакцинальный статус и может давать рекомендации по программам вакцинации на основе литературы.

**Q: Может ли агент работать с лабораторными данными в PDF?**
A: Да, есть инструмент `extract_document` для извлечения текста из PDF и DOCX с OCR.

**Q: Как агент поймет, что это проблема птицы, а не свиней?**
A: При создании чата нужно выбрать агента "VetRetro - Птицеводство" вместо "VetRetro - Свиноводство". Каждый агент специализирован на своей отрасли.
