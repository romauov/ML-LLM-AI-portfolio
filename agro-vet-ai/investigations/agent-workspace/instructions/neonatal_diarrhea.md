# Руководство по расследованию неонатальной диареи у поросят

## Когда использовать эту инструкцию

**Триггеры для начала расследования:**
- Повышенный падеж поросят в возрасте 1-15 дней
- Водянистая диарея у подсосных поросят
- Обезвоживание, истощение новорожденных
- Рост процента потерь на опоросе

**Возрастная группа:** Поросята от 1 до 15 дней жизни

**Цель расследования:** Выявить первопричину диареи (технологическая / инфекционная / комбинированная), определить патоген(ы), разработать план лечения и профилактики.

## Этап 1: Анализ технологических факторов (первичное исключение)

### Цель этапа

Исключить или подтвердить влияние технологических ошибок как первопричины или сопутствующего фактора. Технологические нарушения сами по себе могут вызвать диарею или сделать поросят более восприимчивыми к инфекциям.

### Чек-лист: Прием опороса

- [ ] **Осушение новорожденных:** Используется ли осушитель для поросят сразу после рождения?
- [ ] **Получение молозива:** Получили ли поросята молозиво в первые 1-3 часа жизни?
  - Количество молозива: достаточное / недостаточное / неизвестно
- [ ] **Раздельное вскармливание:** При избытке поросят (более количества сосков) правильно ли организовано раздельное вскармливание?
- [ ] **Обогрев:** Работают ли коврики/лампы локального обогрева?
  - Температура в зоне отдыха поросят: _____ °C (норма: 32-35°C в 1-3 дня, 28-30°C в 4-7 дней)

### Чек-лист: Состояние свиноматки

  - Оценка по шпигомеру (если доступна): _____ мм
- [ ] **Мастит:** Есть ли признаки мастита?
  - Осмотр вымени: болезненность, отек, изменение цвета молока
  - Результат: есть признаки / нет признаков / требуется дополнительный осмотр
- [ ] **Метрит:** Есть ли признаки метрита?
  - Температура тела свиноматки: _____ °C
  - Выделения из влагалища: есть / нет / характер: _____
  - Апатия, отказ от корма: да / нет

### Чек-лист: Условия содержания

- [ ] **Температура:** Соответствует ли температура в секции нормам для возраста?

- [ ] **Вентиляция:** Правильно ли настроена вентиляция?

- [ ] **Кормушки/поилки:** Достаточно ли кормушек и поилок?

  - Доступность: достаточно / недостаточно
- [ ] **Обогащение среды:** Есть ли элементы обогащения (игрушки, цепи, сено)?

### Чек-лист: Технологические аспекты

- [ ] **Плотность посадки:** Нет ли переполнения в станке?

## Ключевые принципы при расследовании неонатальной диареи

1. **Системный подход:** Анализ ведется на уровне группы, а не отдельной особи
2. **Опора на данные:** Все выводы базируются на фактах
3. **Приоритет технологических факторов:** Сначала исключить простые ошибки
4. **Интеграция информации:** Клиника + вскрытие + лаборатория + база знаний
5. **Многофакторность:** Признавать комплексную природу проблемы
6. **Документирование:** Фиксировать каждый этап

## Типичные ошибки при расследовании

❌ Начать лечение антибиотиками без выделения патогена и антибиотикограммы
❌ Игнорировать технологические факторы
❌ Не провести вскрытие павших животных
