# Руководство по расследованию респираторных заболеваний у свиней

## Когда использовать эту инструкцию

**Триггеры для начала расследования:**
- Снижение сохранности на доращивании/откорме
- Рост утилизации "слабых" животных
- Недобор веса к реализации, падение среднесуточных привесов
- Массовый кашель в группах старше 30 дней
- Клинические признаки: кашель, тахипноэ (одышка), истощение, хрипы
- Резкое увеличение применения антибиотиков без стойкого эффекта
- Жалобы переработки на патологию легких (плевриты, абсцессы)

**Возрастная группа:** Любая, но чаще доращивание и откорм (21+ дней)

**Цель расследования:** Выявить критические периоды, определить вероятные причины (инфекционные, технологические, терапевтические), выстроить корректирующие меры.

## Ключевые проверочные листы

### Микроклимат
- [ ] Температура пола/воздуха соответствует нормативам по возрастам
- [ ] Отсутствие сквозняков, особенно в зонах отдыха
- [ ] Влажность 60-75%, аммиак < 20 ppm
- [ ] Своевременная очистка вентиляционных каналов

### Терапия
- [ ] Курсы антибиотиков не прерываются досрочно
- [ ] Дозы соответствуют живой массе животных
- [ ] Используются препараты с доказанной чувствительностью
- [ ] НПВС назначаются при тяжелой клинике

### Лабораторные работы
- [ ] Материал отбирается до начала лечения
- [ ] Изолятам присваиваются ID групп/возрастов
- [ ] Результаты доводятся до ветслужбы и интегрируются в протоколы

### Подсосный период
- [ ] Поросята получают молозиво в первые 1-3 часа
- [ ] Прикорм вводится с 7 дня, контроль поедаемости
- [ ] Своевременная санация свиноматок и контроль маститометритов

## Особые случаи

**Без лабораторных данных:**
- Выводы остаются вероятностными
- Рекомендуется срочный отбор проб при ближайшем падеже
- Использовать патологоанатомические находки

**Резистентность к большинству антибиотиков:**
- Консультация с профильной лабораторией
- Применение комбинированных схем
- Проверка резистентности к препаратам второго ряда

**Подозрение на PRRS/PCV2:**
- При внезапном росте клиники и падеже во всех возрастах
- Запустить отдельное расследование (см. instructions/prrs.md)
- Вакцинация, карантин, биобезопасность
