# Инструкция: Энтеральные проблемы у птицы (диарея, энтериты)

## Когда использовать эту инструкцию

**Триггеры для запуска расследования:**
- Водянистый помет, диарея
- Некротический энтерит (темный, кровянистый помет)
- Снижение привесов при нормальном потреблении корма
- Повышенная смертность без явных респираторных признаков
- Плохая конверсия корма
- Влажная подстилка в зоне клоаки

**Возрастные группы:** Все возраста, но особенно бройлеры 15-35 дней

**Наиболее частые возбудители:**
- Кокцидиоз (Eimeria spp.) - 15-25 дней
- Некротический энтерит (Clostridium perfringens) - 15-35 дней
- E.coli энтерит - любой возраст
- Salmonella spp. - любой возраст
- Ротавирус, реовирус - молодняк 5-15 дней

---

## Системный подход к расследованию

**Принцип:** Технология → Инфекция → Лечение → Профилактика

Энтеральные проблемы у птицы почти всегда МНОГОФАКТОРНЫЕ. Необходимо анализировать:
1. Кормление (корм, кокцидиостатики)
2. Подстилка (влажность, аммиак)
3. Плотность посадки
4. Стресс-факторы
5. Инфекционные агенты

---

## Этап 1: Анализ производственных данных

### Цель
Определить масштаб проблемы и динамику. Выявить корреляции с технологическими факторами.

### Данные для сбора
Запросить у ветеринара:

1. **Динамика падежа и привесов:**
   - Падеж за последние 7-14 дней (голов/день, %/день)
   - Средняя живая масса по возрастам (сравнить с нормой для кросса)
   - Конверсия корма (норма для бройлеров: 1.6-1.9)

2. **Производственный график:**
   - Когда началась проблема? (день возраста стада)
   - Были ли изменения за 3-7 дней до начала? (смена корма, вакцинация, погода)
   - Проблема только в одном птичнике или везде?

3. **Сравнение с другими птичниками:**
   - Если проблема в нескольких → системная причина
   - Если в одном → локальные факторы (микроклимат, кормление, вода)

### Поиск в базе знаний
```
vet_search("enteric diseases broilers age-specific", limit=10)
vet_search("necrotic enteritis Clostridium perfringens poultry", limit=10, source_filter="Practical guide to broiler health management")
vet_search("coccidiosis Eimeria broilers", limit=10)
```

### Результат этапа
- Понимание масштаба и динамики
- Гипотеза о системной vs локальной проблеме
- Предварительный список возможных причин

## Этап 2: Анализ технологии кормления

### Цель
Исключить или подтвердить кормовой фактор как первопричину или усугубляющий элемент.

### Чек-лист кормления

**ВСЕГДА спросить ветеринара:**

1. **История смены корма:**
   - Когда последний раз меняли корм/производителя? (дата)
   - Совпадает ли начало проблемы со сменой корма?
   - Как птица ест корм? (нормально / плохо / отказывается)

2. **Кокцидиостатики:**
   - Содержит ли корм кокцидиостатик? Какой? В какой концентрации?
   - Если да → правильная ли концентрация? Эффективен ли препарат?
   - Если нет → высокий риск кокцидиоза!

3. **Качество корма:**
   - Визуальная оценка: плесень, слежался, влажный?
   - Запах: затхлый, кислый, гнилостный?
   - Хранение: правильные условия (сухое, проветриваемое)?

4. **Состав корма (если доступно):**
   - Протеин для фазы: соответствует ли норме?
   - Жиры: не прогорклые ли?
   - Влажность: <13%?

### Когда запрашивать анализ корма

**Анализ на микотоксины ОБЯЗАТЕЛЕН, если:**
- Корм хранился неправильно (влажность, плесень)
- Визуально заметна плесень
- Высокая смертность без явной инфекции
- Симптомы интоксикации (угнетение, отказ от корма)

**Стандартный анализ корма:**
- Микотоксины (афлатоксины >20 ppb критично для птицы!)
- Протеин, жиры, клетчатка
- Влажность

```
vet_search("feed quality broilers diarrhea", limit=10)
vet_search("anticoccidial drugs poultry", limit=10)
```

### Результат этапа
- Кормовой фактор исключен ИЛИ идентифицирован
- Решение о необходимости анализа корма
- Оценка программы профилактики кокцидиоза

## Этап 3: Оценка условий содержания

### Цель
Выявить стресс-факторы и условия, способствующие развитию энтеральных проблем.

### Чек-лист микроклимата и содержания

**Запросить у ветеринара (или создать полевой чек-лист):**

1. **Подстилка:**
   - Состояние: сухая / влажная / мокрая / комковатая?
   - Наличие корки? (способствует росту кокцидий)
   - Последняя подсыпка? Частота подсыпки?

2. **Качество воздуха:**
   - Запах аммиака: есть / нет / раздражает глаза?
   - Оценка: <20 ppm - норма, >25 ppm - критично
   - Аммиак → раздражение кишечника → энтерит

3. **Плотность посадки:**
   - Текущая плотность (кг/м² или голов/м²)
   - Норма для бройлеров: 30-40 кг/м²
   - Высокая плотность → стресс → снижение иммунитета → энтериты

4. **Водоснабжение:**
   - Птица пьет нормально?
   - Соотношение вода/корм (норма: 1.8-2.2)
   - Низкое потребление воды → обезвоживание при диарее

5. **Температура и вентиляция:**
   - Соответствует ли норме для возраста?
   - Резкие колебания температуры → стресс → энтериты

### Создать полевой чек-лист `03a_field_checklist.md`

Если ветеринар планирует выезд, создать готовый к печати чек-лист (см. пример в AGENTS_POULTRY.md, раздел "Предоставлять чек-листы для полевой работы").

```
vet_search("ammonia poultry intestinal health", limit=10)
vet_search("stocking density welfare broilers", limit=10)
```

### Результат этапа
- Список стресс-факторов и способствующих условий
- Рекомендации по коррекции технологии
- Готовый чек-лист для полевого осмотра

## Этап 4: Клинические данные и вскрытие

### Цель
Собрать клинические признаки для дифференциальной диагностики.

### Клинические признаки

**Запросить у ветеринара (или провести осмотр):**

1. **Общее состояние птицы:**
   - Активность: вялость, угнетение?
   - Аппетит: нормальный / сниженный / отказ?
   - Оперение: взъерошенное, грязное в области клоаки?

2. **Характер помета:**
   - Водянистый (кокцидиоз, вирусы, E.coli)
   - Кровянистый, темный, дегтеобразный (некротический энтерит, Eimeria necatrix/tenella)
   - Зеленый, пенистый (Salmonella, E.coli)
   - Оранжевый с примесью крови (E.coli, кокцидиоз)
   - Непереваренный корм (мальабсорбция, энтериты)

3. **Процент пораженной птицы:**
   - Единичные случаи vs массовая проблема

### Вскрытие (патологоанатомические признаки)

**КРИТИЧНО для диагностики энтеральных проблем!**

Запросить у ветеринара описание вскрытия (или создать чек-лист):

1. **Кишечник:**
   - Вздутие, газы в кишечнике?
   - Содержимое: водянистое, кровянистое, слизистое?
   - Стенка кишечника: утолщена, гиперемирована, геморрагии?
   - Некротические бляшки на слизистой? (некротический энтерит)
   - Характерная локализация поражения (тонкий/толстый кишечник, слепые отростки)

2. **Кокцидиоз - специфические признаки:**
   - **E.acervulina**: белые полосы на тонком кишечнике (легкая форма)
   - **E.maxima**: геморрагии в средней части тонкого кишечника, слизь
   - **E.tenella/necatrix**: кровь в слепых отростках, некроз, темное содержимое

3. **Некротический энтерит - специфические признаки:**
   - Некротические бляшки/пленки на слизистой тонкого кишечника
   - Дифтеритическое воспаление
   - Дряблая, истонченная стенка кишечника
   - Характерный запах

4. **Другие органы:**
   - Печень: увеличена, геморрагии, некрозы? (Salmonella, E.coli)
   - Селезенка: увеличена? (Salmonella)

### Поиск в базе знаний

```
vet_search("coccidiosis Eimeria species lesions chickens", limit=10)
vet_search("E.coli colibacillosis poultry pathology", limit=10)
vet_search("Salmonella enteritis chickens diagnosis", limit=10)
```

## Этап 5: Лабораторная диагностика

### Цель
Подтвердить возбудителя и определить чувствительность к препаратам.

### Необходимые исследования

**Приоритет 1 (КРИТИЧНО):**

1. **Микроскопия помета на кокцидии:**
   - Материал: свежий помет от больных птиц
   - Цель: обнаружение ооцист Eimeria, определение видового состава
   - Результат за 1-2 часа
   - **САМЫЙ БЫСТРЫЙ метод диагностики кокцидиоза!**

2. **Бактериологический посев содержимого кишечника:**
   - Материал: слизистая тонкого кишечника, слепые отростки
   - Цель: выявление E.coli, Salmonella, Clostridium perfringens
   - + Антибиотикограмма (критично для лечения!)

**Приоритет 2 (ЖЕЛАТЕЛЬНО):**

3. **ПЦР на кокцидиоз** (если микроскопия отрицательна, но клиника типична)

4. **ПЦР на Clostridium perfringens + определение токсина alpha:**
   - Материал: содержимое тонкого кишечника
   - Важно: сам Cl.perfringens - нормальная флора, важен ТОКСИН!

5. **ПЦР на ротавирус, реовирус** (если молодняк <14 дней)

6. **Вирусологическое исследование** (если подозрение на вирусный энтерит)

### Отбор материала

**Важные правила:**
- Отбирать от свежепавших или эвтаназированных БОЛЬНЫХ птиц
- Кишечник - целый участок с содержимым (10-15 см)
- Для микроскопии - свежий помет (не старше 2 часов)
- Для бактериологии - стерильные контейнеры, доставка в течение 4 часов или замороженным

## Этап 6: Формирование гипотез

### Цель
На основе всех собранных данных ранжировать гипотезы по вероятности.

### Принцип формирования гипотез

**Учитывать:**
1. Возраст птицы (разные возбудители в разном возрасте)
2. Клинику + патологоанатомию
3. Технологические факторы (корм, подстилка, плотность)
4. Лабораторные данные (если есть)

**Типичные гипотезы:**

#### Гипотеза 1: Кокцидиоз (+ вторичные инфекции)
**Уверенность:** high / medium / low (в зависимости от данных)

**Аргументация:**
- Возраст: 15-25 дней (типичный возраст для кокцидиоза)
- Клиника: кровянистый/водянистый помет, снижение привесов
- Патологоанатомия: поражение кишечника (указать локализацию и вид Eimeria)
- Лабораторные данные: ооцисты в помете / ПЦР положительно
- Технология: отсутствие кокцидиостатиков в корме ИЛИ влажная подстилка

**Источники:**
- [Practical guide to broiler health management], стр. XX: "Кокцидиоз - основная причина энтеритов у бройлеров 14-28 дней..."

#### Гипотеза 2: Некротический энтерит (Clostridium perfringens)
**Уверенность:** high / medium / low

**Аргументация:**
- Возраст: 15-35 дней
- Клиника: темный, дегтеобразный помет, внезапная смертность
- Патологоанатомия: некротические бляшки в тонком кишечнике
- Факторы предрасположенности: кокцидиоз (повреждение слизистой) + высокопротеиновый корм
- Лабораторные данные: Cl.perfringens + токсин alpha

**Источники:**
- [Practical guide to broiler health management], стр. YY: "Некротический энтерит часто развивается на фоне кокцидиоза..."

#### Гипотеза 3: Колибактериоз (E.coli)
**Аргументация:**
- Любой возраст
- Часто вторичная инфекция на фоне стресса (кокцидиоз, плохой микроклимат)
- Бактериологический посев: E.coli в высоком титре

## Этап 7: Рекомендации по лечению и профилактике

### Цель
Предоставить конкретные, действенные рекомендации с учетом антибиотикограмм и результатов расследования.

### Структура рекомендаций

#### 7.1. Срочные меры (немедленно)

**Если подтвержден кокцидиоз:**
- Добавить кокцидиостатик в корм ИЛИ провести лечение водорастворимым кокцидиоцидом
  - Примеры: ампролиум, толтразурил, сульфаниламиды
  - Дозировки: искать в базе знаний с учетом возраста и тяжести

**Если подтвержден некротический энтерит:**
- Антибиотикотерапия (пенициллины эффективны против Cl.perfringens)
  - Амоксициллин, ампициллин (дозировки для птицы!)
  - Курс: 5-7 дней
  - Путь введения: с водой (групповое лечение)

**Если подтвержден колибактериоз:**
- Антибиотикотерапия по результатам антибиотикограммы
  - Часто эффективны: фторхинолоны (энрофлоксацин), цефалоспорины, колистин
  - Курс: 5-7 дней

**Поддерживающая терапия (для всех случаев):**
- Витамины (A, E, C) для поддержки иммунитета и регенерации слизистой
- Пробиотики для восстановления нормальной флоры
- Органические кислоты в воду (снижение pH, улучшение пищеварения)

### Поиск в базе знаний для доз и режимов
```
vet_search("amprolium dosage broilers coccidiosis", limit=10, source_filter="Practical guide to broiler health management")
vet_search("amoxicillin necrotic enteritis chickens dose", limit=10, source_filter="Antimicrobial Therapy in Veterinary Medicine")
vet_search("enrofloxacin E.coli poultry dosage", limit=10)
```

#### 7.2. Технологические корректировки (в течение 1-3 дней)

1. **Подстилка:**
   - Немедленная подсыпка сухой подстилки
   - Улучшение вентиляции для снижения влажности
   - Контроль протечек поилок

2. **Кормление:**
   - Если корм подозрительный → заменить
   - Добавить кокцидиостатик (если не было)
   - Рассмотреть добавку ферментов (улучшение переваримости)

3. **Микроклимат:**
   - Снизить аммиак (<20 ppm): улучшить вентиляцию + подсыпка
   - Стабилизировать температуру (избегать резких колебаний)

4. **Плотность:**
   - Если возможно, снизить плотность посадки (убрать часть поголовья)

#### 7.3. Профилактика для следующих туров

1. **Программа профилактики кокцидиоза:**
   - Ротация кокцидиостатиков (избежать резистентности)
   - Соблюдение санитарного разрыва (минимум 14 дней "пусто")
   - Дезинфекция с кокцидиоцидным действием

2. **Биобезопасность:**
   - Принцип "все пусто - все занято"
   - Контроль качества корма (анализ на микотоксины 1-2 раза в год)
   - Улучшение условий хранения корма

3. **Мониторинг:**
   - Еженедельный контроль привесов и конверсии
   - Ежедневный контроль падежа
   - Регулярные вскрытия павшей птицы
   - Профилактические микроскопии помета на кокцидии (каждые 10-14 дней)

### Результат этапа
- Файл `08_conclusions.md` с конкретными рекомендациями
- Разделение на срочные меры, технологические корректировки, долгосрочную профилактику

## Этап 8: Финальный отчет и мониторинг

### Цель
Завершить расследование и предоставить полный отчет.

### Структура финального отчета `09_report.md`

1. **Резюме случая**
2. **Проведенный анализ** (этапы 1-7)
3. **Результаты лабораторных исследований**
4. **Основная гипотеза** (с обоснованием)
5. **Рекомендации** (лечение + технология + профилактика)
6. **Прогноз** (ожидаемое улучшение через X дней)
7. **Критерии эффективности мер:**
   - Снижение падежа до <X%/день
   - Улучшение конверсии корма
   - Нормализация помета

### Мониторинг эффективности

**Попросить ветеринара отслеживать:**
- Падеж (ежедневно) → должен снизиться через 3-5 дней
- Привесы (еженедельно) → должны восстановиться через 7-10 дней
- Конверсия корма → должна улучшиться через 7-14 дней
- Клиника → помет должен нормализоваться через 5-7 дней

**Если улучшения НЕТ через 7 дней:**
- Пересмотреть гипотезы
- Дополнительные лабораторные исследования

## Часто встречающиеся ошибки

❌ **НЕ ДЕЛАТЬ:**
- Ставить диагноз "кокцидиоз" без микроскопии помета или патологоанатомии
- Назначать антибиотики без антибиотикограммы (риск резистентности)
- Игнорировать технологические факторы (подстилка, аммиак, плотность)
- Лечить только антибиотиками без коррекции условий содержания

✅ **ДЕЛАТЬ:**
- ВСЕГДА проводить вскрытие при энтеральных проблемах
- Микроскопия помета - быстрый и доступный метод
- Учитывать многофакторность проблемы
- Корректировать технологию параллельно с лечением

## Ключевые моменты для птицеводства

1. **Кокцидиоз - основная причина энтеритов у бройлеров 15-28 дней**
2. **Некротический энтерит часто вторичен после кокцидиоза**
3. **Влажная подстилка = кокцидиоз + дерматит лап**
4. **Микроскопия помета - самый быстрый метод диагностики кокцидиоза**
5. **Профилактика эффективнее лечения: кокцидиостатики в корме + сухая подстилка**
