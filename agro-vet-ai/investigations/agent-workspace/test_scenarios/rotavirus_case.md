# Тестовый сценарий: Ротавирусная инфекция у поросят

## Тип проблемы
Неонатальная диарея (вирусная этиология)

## Ключевые отличия от предыдущего кейса (E.coli)
- Вирусная инфекция (антибиотики неэффективны)
- Характерный беловато-желтый налет на слизистой при вскрытии
- ПЦР положительный на ротавирус
- Требуется другая вакцина для свиноматок
- Меньшая летальность при правильной регидратации
